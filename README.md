

## Setup

1. Clone this repository:
   ```bash
   git clone https://github.com/your-username/react-candidate-test.git
   cd react-candidate-test
   ```

2. Install dependencies:
   ```bash
   npm install
   ```
   or
   ```bash
   yarn install
   ```

3. Start the development server:
   ```bash
   npm start
   ```
   or
   ```bash
   yarn start
   ```
